# 🌦️ Weather-Related Disease Predictor

This is a Machine Learning web application that predicts the likelihood of various diseases based on weather conditions (Temperature, Humidity, Wind Speed) and physical symptoms.

## 🚀 Features
- **High Accuracy:** The model achieves approximately **98% accuracy** using a Gradient Boosting Classifier.
- **Interactive UI:** Built with Streamlit for an easy-to-use web interface.
- **Comprehensive Analysis:** Evaluates 45 different features including age, gender, and specific symptoms like nausea, joint pain, and fatigue.

## 🛠️ Tech Stack
- **Language:** Python
- **ML Framework:** Scikit-learn (Gradient Boosting)
- **Web App:** Streamlit
- **Environment:** Trained in Google Colab, deployed on GitHub.

## 📋 Diseases Predicted
The model can identify risks for:
- Arthritis, Common Cold, Dengue, Eczema, Heart Attack, Heat Stroke, Influenza, Malaria, Migraine, Sinusitis, and Stroke.

## ⚙️ How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
